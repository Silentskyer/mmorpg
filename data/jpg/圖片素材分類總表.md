# 圖片素材分類總表

目前 `data/jpg` 內共有 `402` 張圖片，已重新整理成可直接對應遊戲系統的資料夾結構。

## 主要分類
- `equipment/weapon`
  武器圖示，已依 `axe / book / bow / dagger / fist / gun / mace / spear / sword / throwing` 分類。
- `equipment/armor`
  防具圖示，已分成 `body / feet / head`。
- `equipment/accessory`
  飾品圖示，已分成 `medal / necklace / ring`。
- `items/consumable`
  消耗品圖示，已分成 `food / misc / potion / remedy`。
- `items/material`
  素材圖示，已分成 `mineral / organic`。
- `items/currency`
  貨幣圖示。
- `items/key`
  鑰匙類圖示。
- `items/utility`
  文件、工具、寶箱等功能型道具圖示。
- `skills/element`
  屬性技能圖示，已分成 `earth / fire / holy / ice / poison / shadow / thunder / water / wind`。
- `skills/physical`
  物理技能圖示，已分成 `bow / impact / sword`。
- `skills/support`
  輔助技能圖示，已分成 `arcane / buff / death / light`。
- `environment/resource`
  環境與資源圖示，已分成 `bones / gold / metal / wood`。

## 建議綁定方式
- 裝備欄位圖示優先使用 `equipment/`
- 背包、掉落與素材欄位優先使用 `items/`
- 技能按鈕與技能樹節點優先使用 `skills/`
- 地區素材、採集物與場景點綴可使用 `environment/`
